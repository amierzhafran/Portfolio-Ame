# AME Portfolio Website

A simple static portfolio website built with HTML and CSS.

## Project Overview

This site appears to be a personal portfolio landing page for "AME".
It includes a homepage, an about page, a resume page, and a special "Simply Lovely" page.

## Files and Structure

- `index.html` - Main landing page with hero section and navigation.
- `about.html` - About Me page.
- `resume.html` - Resume page.
- `loml.html` - "Simply Lovely" page with a personal message.
- `css/styles.css` - Shared site styles.
- `images/` - Image assets used by the site.

## How to Use

1. Open `index.html` in a web browser.
2. Navigate between pages using the top navigation menu.

## Notes

- The project is static and requires no build tools.
- Images are loaded from the `images/` directory.
- Google Fonts are used for typography.

## Customization

- Update text content in the HTML files to personalize the site.
- Replace images in `images/` and update `src` paths as needed.
- Change colors or layout in `css/styles.css`.
